<objective>
Build the WebM recording tier of EXPT-02: a pure, mock-testable recording pipeline (`record-replay.ts`) plus the React glue (`use-record-replay.ts`) and confirm modal that turn the ExportMenu "Record replay as video" item into a one-click scripted recording — enter replay, seek 0, record the active panel at 2x while auto-playing at the current speed, auto-stop at the last event, and download a `.webm`, with codec feature-detection, a duration-estimate confirm, and abort-on-hidden.

Output: `record-replay.ts` + `use-record-replay.ts` + `RecordConfirmModal.tsx`, wired into SessionDetails via the ExportMenu Record item, with unit tests mocking MediaRecorder/captureStream/requestFrame/html2canvas.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| replay frames → in-memory WebM blob | A long/slow replay accumulates an unbounded blob in client memory |
| tab visibility → recorder clock | rAF throttling on a hidden tab freezes frames while the recorder clock runs |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-15 | Denial of Service (client memory) | unbounded WebM blob on a huge/slow replay | mitigate | D-26 duration-estimate confirm (>120s); controller Stop discards (D-23); abort-on-hidden (D-24) — three brakes, no hard cap by design |
| T-02-16 | Tampering (corrupt capture) | hidden-tab rAF throttle freezing frames while the clock runs | mitigate | visibilitychange→hidden aborts + discards the partial (D-24) rather than producing a frozen-frame video |
| T-02-17 | Denial (cross-browser) | assuming MediaRecorder presence implies WebM support (Safari) | mitigate | pickMimeType null ⇒ disable the video option + tooltip (D-25); never construct a recorder without a supported codec |
| T-02-18 | Resource (client) | recording blob/object-URL leak | mitigate | object-URL revoked in finally (export-png idiom); discard() frees chunks |
| T-02-SC | Tampering | npm installs | accept | No new packages (native MediaRecorder/captureStream + existing html2canvas); legitimacy audit: none flagged |
</threat_model>
